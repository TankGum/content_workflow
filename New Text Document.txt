## 1. Tổng Quan

Hệ thống được thiết kế để sản xuất video ngắn (Shorts/Reels/TikTok) tự động dựa trên 2 chiến lược nội dung:

| Đặc điểm | Channel 1: Trending | Channel 2: Evergreen |
| :--- | :--- | :--- |
| **Mục tiêu** | Tin hot, sự kiện, AI, công nghệ, drama | Lịch sử, Khoa học, Vũ trụ, Giáo dục |
| **Tuổi thọ video** | Ngắn hạn (Viral nhanh) | Dài hạn (Evergreen, tồn tại nhiều năm) |
| **Tần suất** | 3 video / tuần | 1–2 video / tuần |
| **Yêu cầu chính** | Tốc độ xử lý nhanh | Độ chính xác thông tin (Fact-check) |
| **Ví dụ** | "AI just replaced programmers?" | "What killed the dinosaurs?" |

Channel 1: Trending Content Pipeline
Luồng xử lý (Workflow)
Collect Trends: Lấy dữ liệu từ Google Trends, Reddit, News API.
Rank Topic: AI chọn topic có tiềm năng viral nhất.
Generate Script: Viết kịch bản 40s (Hook, Body, Ending).
AI Review: Agent khác kiểm tra độ hấp dẫn.
Media Gen: Tạo Voice (ElevenLabs) và Image (Stable Diffusion).
Render: Ghép video (FFmpeg/MoviePy).
Publish: Đẩy lên nền tảng.
Code Implementation (Worker)

# app/workers/trending_pipeline.py
from celery import Celery
from app.channels.trending import collector, ranker, scripter, media, renderer

app = Celery('tasks', broker='redis://localhost:6379/0')

@app.task(bind=True, max_retries=3)
def run_trending_pipeline(self):
    try:
        # Step 1: Collect Trends
        raw_topics = collector.fetch_trends(sources=['google', 'reddit', 'news_api'])
        
        # Step 2: Rank Topics (AI)
        best_topic = ranker.select_viral_topic(topics=raw_topics)
        
        # Step 3: Generate Script
        script_data = scripter.generate_script(topic=best_topic)
        
        # Step 4: AI Review (Critique)
        improved_script = scripter.review_script(script=script_data)
        
        # Step 5: Generate Media (Async parallel)
        voice_path = media.generate_voice(text=improved_script['narration'])
        images = media.generate_images(prompts=improved_script['image_prompts'])
        
        # Step 6: Render Video
        video_path = renderer.render_short(
            audio=voice_path, 
            images=images, 
            subtitles=improved_script['subtitles']
        )
        
        # Step 7: Publish
        publisher.upload(video_path, platforms=['tiktok', 'youtube_shorts'])
        
        return {"status": "success", "video": video_path}
        
    except Exception as exc:
        raise self.retry(exc=exc, countdown=300)

5. Channel 2: Evergreen Content Pipeline
Luồng xử lý (Workflow)
Topic Library: Chọn topic từ DB (chưa làm).
Research Agent: Tổng hợp thông tin từ Wikipedia, NASA, Science Articles.
Script Generator: Viết kịch bản giáo dục.
Fact-Check Agent: Verify độ chính xác (Critical Step).
Media & Render: Tương tự Channel 1 nhưng chất lượng hình ảnh cao hơn.
Code Implementation (Worker)

# app/workers/evergreen_pipeline.py
from app.channels.evergreen import library, research, scripter, factcheck

@app.task(bind=True)
def run_evergreen_pipeline(self, topic_id=None):
    # Step 1: Select Topic from DB
    if not topic_id:
        topic = library.get_unused_topic(category='space')
    else:
        topic = library.get_topic(topic_id)
    
    # Step 2: Deep Research (RAG or Search)
    research_data = research.gather_facts(topic=topic.name, sources=['wikipedia', 'nasa'])
    
    # Step 3: Generate Script
    script = scripter.create_educational_script(topic=topic, facts=research_data)
    
    # Step 4: Fact-Check Agent (Critical Step)
    is_accurate = factcheck.verify(script=script, source_data=research_data)
    
    if not is_accurate:
        # Loop back to Step 3 if facts are wrong
        return run_evergreen_pipeline.retry(kwargs={'topic_id': topic.id})
    
    # Step 5, 6, 7, 8: Same as Channel 1 (Voice, Visual, Render, Publish)
    return produce_final_video(script)

7. Cơ Sở Dữ Liệu (Database Schema)
Sử dụng SQLAlchemy Models.
# app/models.py
from sqlalchemy import Column, Integer, String, Text, DateTime, Boolean
from sqlalchemy.ext.declarative import declarative_base

Base = declarative_base()

class VideoProject(Base):
    __tablename__ = "video_projects"
    
    id = Column(Integer, primary_key=True)
    channel_type = Column(String) # 'trending' or 'evergreen'
    topic = Column(String)
    script = Column(Text)
    status = Column(String) # 'pending', 'processing', 'done', 'failed'
    video_url = Column(String)
    fact_check_passed = Column(Boolean, default=False)
    created_at = Column(DateTime)
    updated_at = Column(DateTime)

class TopicLibrary(Base):
    __tablename__ = "topic_library"
    
    id = Column(Integer, primary_key=True)
    name = Column(String)
    category = Column(String) # 'history', 'space', 'science'
    used = Column(Boolean, default=False)

9. Best Practices & Khuyến Nghị
1. Quản lý State (Trạng thái)
Mỗi bước trong pipeline phải lưu trạng thái vào DB.
Trạng thái: PENDING -> SCRIPT_DONE -> MEDIA_DONE -> PUBLISHED.
Nếu bước "Render Video" lỗi, không cần chạy lại từ đầu.
2. Xử lý lỗi (Error Handling)
API AI có thể fail. Luôn có cơ chế Retry (thử lại) với exponential backoff.
Nếu AI generate ảnh lỗi quá 3 lần -> Chuyển sang ảnh stock (Pexels/Unsplash).
3. Tối ưu chi phí
Channel 1: Dùng model nhanh/rẻ (gpt-3.5-turbo / Haiku) cho ranking/script.
Channel 2: Dùng model xịn (gpt-4o / Sonnet) cho Research và Fact-check.
4. Storage
Lưu file âm thanh/video tạm vào AWS S3 hoặc Google Cloud Storage.
Không lưu trực tiếp trên server app để dễ scale (horizontal scaling).
5. Monitoring
Dùng Prometheus + Grafana hoặc log vào ELK Stack.
Theo dõi: Số lượng video thành công, chi phí API/token, thời gian xử lý trung bình.
6. Security
Quản lý API Keys qua Environment Variables (.env).
Không commit file .env vào Git.
